# Demo assets

This directory is reserved for future screenshots, gifs, or terminal recordings used by the main project README.

Recommended assets to add:

1. `menu.png`
   - `git menu` with fzf preview panel open
2. `recent-branch-picker.png`
   - `git recent` showing branch metadata and log preview
3. `worktree-picker.png`
   - `git wttmux` or `git wt path` picker with status/log/file preview
4. `lazygit-workflow.png`
   - lazygit opened inside a feature worktree
5. `tmux-cockpit.png`
   - tmux session showing shell, git, and AI windows

Suggested capture guidelines:

- Use a clean terminal theme with good contrast
- Keep repo names and branch names realistic
- Avoid personal/private remotes or secrets in the capture
- Prefer 1600x900 or similar wide layout for terminal screenshots
- If you record gifs, keep them short and task-focused
