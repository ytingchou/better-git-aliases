# better-git-aliases v0.5.2

This patch release improves two practical areas:

- workspace intelligence
- roadmap and provider support clarity

## Highlights

- `git wsdash`
  - now shows workspace attention counts and prioritizes repos that are dirty or need upstream attention
- `git wsrecent`
  - gives you a fast way to jump to the most relevant recent repo in a workspace
- roadmap and provider docs
  - the project now has clearer portfolio planning and a capability matrix for host-aware features
