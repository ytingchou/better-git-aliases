# better-git-aliases v0.5.3

This patch release rounds out the first remote review workflow.

## Highlights

- `git remote-menu`
  - provides one focused entrypoint for remote status, repo opening, compare, info, and provider capability checks
- `git remote-capabilities`
  - shows current provider support and fallback behavior without making you open the docs first
- smarter `git remote-status`
  - now recommends the next action based on upstream state and provider support
- better compare flow
  - in interactive mode, `remote-menu` lets you override base and compare branches before opening compare URLs
