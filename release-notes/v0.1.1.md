# better-git-aliases v0.1.1

Small documentation-focused patch release.

## Highlights

- Added a beginner-friendly day-to-day tutorial
- Added a practical lazygit onboarding guide and cheatsheet
- Removed hard-coded local home directory paths from Markdown docs
- Improved generic shell integration examples in the README

## Good for

- Users who are new to `lazygit`
- Users who want a simpler daily workflow guide
- Users reading the repo on GitHub and expecting portable examples

## Suggested starting point

```bash
git doctor
git menu
git feature start my-task
gwt my-task
git lg
```
