# better-git-aliases v0.1.0

Initial release of a practical Git CLI workflow toolkit focused on real day-to-day engineering work.

## Highlights

- Short, discoverable Git aliases
- Interactive `fzf` menu, branch picker, worktree picker, and commit browser
- First-class `git worktree` workflow for parallel feature development
- `lazygit`, `tmux`, `codex`, and `cline` integration
- Shell helpers for jumping into worktrees in the current shell
- Built-in `doctor`, `bootstrap`, and `lazygit-config` commands
- Installer and safe uninstaller
- README with setup guide, usage tutorial, demo flow, and lazygit workflow guide

## Included in this release

- `bga` core command launcher
- Git global alias include file
- Curated lazygit config
- Worktree-oriented feature workflow
- Demo asset guidance and publish-ready repo metadata

## Quick start

```bash
./install.sh
git doctor
git menu
git feature start my-feature
gwt my-feature
git cockpit
```
